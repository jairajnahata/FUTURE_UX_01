**Sri Sai Men's Hair Saloon — Website Redesign
Future Interns, UI/UX Design, Task 1**


**About The Project :**

This is my Task 1 submission for the Future Interns UI/UX 
Design Internship. I redesigned a website for a local mens 
hair saloon called Sri Sai Mens Hair Saloon based in 
Hyderabad, India.

The main focus was on making the website look good, feel 
easy to use and actually help the business get more 
bookings and walk ins.


**Pages Built :**

Homepage — Hero section, services preview, testimonials 
and booking call to action.

Services Page — Full service list with images and pricing.

Contact Page — Booking form, map placeholder and 
opening hours.


**Features :**

Clear call to action buttons throughout the site.
Service cards with images and pricing.
Customer reviews section with carousel.
Appointment booking form with time slot selection.
Opening hours table with Tuesday marked as Holiday.
Fully mobile responsive layout.
Smooth animations on scroll and hover.


**Tools and Technologies :**
Figma AI was used for design and prototyping.
React.js was used for building the website.
Framer Motion was used for animations.
Tailwind CSS was used for styling.
GitHub was used for storing the code.


**Project Structure :**
app folder contains the main pages and components.
imports folder contains the dependencies.
styles folder contains the styling files.
